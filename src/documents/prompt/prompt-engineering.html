Strong Prompt Example

Same Task, Engineered.

Clear Role

"You are a technical summarizer"
"Act as a senior software architect"
"You are a friendly customer support agent"

Defined Task

"Extract key points from documents"
"Debug the code and suggest fixes"
"Write engaging social media captions"

Output Structure

"Provide 3-5 bullet points, max 20 words each"
"Format as JSON with fields: title, body, tags"
"Return a table with columns: date, amount, category"

Constraints

"Use only information from the document"
"Do not include any placeholders or lorem ipsum"
"Avoid technical jargon unless explained"

Failure Handling

"If document is unclear, state what's missing"
"If unsure, say 'I don't know' rather than guess"
"Flag any assumptions with a [?] marker"
This is prompt engineering, not just rewriting.
